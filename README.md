I'm a **cybersecurity learner** turning into a **penetration tester**, focused on practical offensive security rather than just theory.

Instead of stopping at tutorials, I run my own **vulnerable VM labs**, work through **CTFs**, and document the full path from **recon → exploitation → privilege escalation → reporting**.

### What I bring

- Strong foundation in **Linux, networking, and web application fundamentals**
- Hands-on exploitation practice across **OWASP Top 10** vulnerabilities
- Practical workflow thinking around **reconnaissance, scanning, and reporting**
- A builder mindset combining **security + scripting + clear documentation**
- Write-ups that help reviewers quickly understand the vulnerability and its fix
